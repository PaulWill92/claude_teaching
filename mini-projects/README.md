# Mini Projects

Five beginner projects in roughly increasing difficulty. Each folder has a `LESSON.md` (read this first) and a `.rb` starter file (do the work here).

| # | Project | Key concepts |
|---|---------|-------------|
| 01 | Number Guessing Game | `rand`, `while` loop, `.to_i`, counters |
| 02 | To-Do List | arrays, `loop/break`, `each_with_index`, `case/when` |
| 03 | Calculator | `def` methods, return values, `.to_f`, edge cases |
| 04 | Word Counter | string methods, hashes, `max_by`, `sort_by`, blocks |
| 05 | Contact Book | array of hashes, `select`/`find`/`delete_if`, symbols |

## How to work through a project

1. Read `LESSON.md` fully before touching the `.rb` file.
2. Open the `.rb` file — each `STEP` comment tells you what to build next.
3. Run it often: `ruby <filename>.rb` — don't wait until everything is done.
4. Once the core version works, pick a stretch goal from the lesson.

## Running a project

```bash
cd mini-projects/01_number_guessing_game
ruby number_guessing_game.rb
```
